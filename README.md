# 📖 Vet-clinic <a name="about-project"></a>



**Vet-clinic** is a relational database to create the data structure for a vet clinic.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

<details>
<summary>Database</summary>
  <ul>
    <li><a href="https://www.postgresql.org/">PostgreSQL</a></li>
  </ul>
</details>

<!-- Features -->

### Key Features <a name="key-features"></a>

- Create animals table, query and update animals table.
- Query multiple tables, add "join table" for visits.
- Database performance audit, add database schema diagram.
